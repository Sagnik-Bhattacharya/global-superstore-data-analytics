## Objective
Perform exploratory data analysis on the Global Superstore dataset to uncover sales, profit, and customer behavior insights.

## Dataset
- Orders across multiple countries, regions, and product categories
- Includes sales, profit, discounts, shipping, and customer segments

## Key Steps
- Data cleaning and preprocessing
- Feature understanding and transformation
- Exploratory Data Analysis (EDA)
- Business insights and recommendations

## Key Insights
- Sales growth does not always translate into profit growth
- High discounts significantly reduce profitability
- Furniture category underperforms in profit
- Same Day shipping reduces margins
- A small set of customers contributes disproportionately to revenue
